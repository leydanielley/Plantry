# Release Notes v0.8.0 - Play Store

## English

**Version 0.8.0 - Advanced RDWC Logging**

NEW FEATURES:
• Hydro Buddy style nutrient logging - track ml amounts and see PPM/EC contribution per fertilizer in real-time
• 4 specialized log types with smart forms:
  - Quick Measurement: Fast daily value logging
  - Water Addback: Track refills with nutrient details
  - Full Reservoir Change: Document complete water changes with cleaning checklist
  - Maintenance: System upkeep checklist for pumps, filters, tubes & more
• Expandable log entries showing detailed nutrient breakdown
• Real-time PPM/EC calculation as you add fertilizers
• Individual fertilizer contributions displayed with visual indicators

IMPROVEMENTS:
• Modern UI with improved navigation
• Dynamic forms that adapt to your needs
• Better visual hierarchy with icons and colors
• Enhanced code quality

All your existing features remain available!

• Echtzeit-PPM/EC-Berechnung beim Hinzufügen von Düngern

VERBESSERUNGEN:
• Modernes UI mit verbesserter Navigation

Alle bestehenden Funktionen bleiben erhalten!
