# So lädst du die Privacy Policy hoch

## Option 1: GitHub Pages (EMPFOHLEN - Kostenlos & Einfach)

### Schritt 1: Repository erstellen
1. Gehe zu https://github.com und logge dich ein
2. Klicke auf "+" → "New repository"
3. Name: `plantry-privacy` (oder beliebig)
4. Wähle: ✅ Public
5. Klicke "Create repository"

### Schritt 2: Datei hochladen
1. Im neuen Repository, klicke "Add file" → "Upload files"
2. Ziehe die Datei `privacy-policy-template.html` in den Browser
3. **WICHTIG:** Benenne sie um zu `index.html` (oder lass sie als `privacy-policy-template.html`)
4. Klicke "Commit changes"

### Schritt 3: GitHub Pages aktivieren
1. Gehe zu "Settings" im Repository
2. Scrolle runter zu "Pages" (linke Seitenleiste)
3. Unter "Source": Wähle "main" branch
4. Klicke "Save"
5. Warte 1-2 Minuten

### Schritt 4: URL kopieren
Deine Privacy Policy ist jetzt verfügbar unter:
```
https://deinusername.github.io/plantry-privacy/privacy-policy-template.html
```
oder (falls du zu index.html umbenannt hast):
```
https://deinusername.github.io/plantry-privacy/
```

## Option 2: Google Sites (Kostenlos & Sehr Einfach)

### Schritt 1: Google Site erstellen
1. Gehe zu https://sites.google.com
2. Klicke auf "+ Blank" (neue leere Site)
3. Titel: "Plantry Privacy Policy"

### Schritt 2: Inhalt einfügen
1. Kopiere den HTML-Inhalt aus `privacy-policy-template.html`
2. Füge den Text-Teil (ohne HTML-Tags) in Google Sites ein
3. Formatiere mit den Google Sites Tools

### Schritt 3: Veröffentlichen
1. Klicke oben rechts auf "Publish"
2. Wähle eine URL (z.B. `plantry-privacy`)
3. Klicke "Publish"
4. Kopiere die URL

**Fertige URL:**
```
https://sites.google.com/view/plantry-privacy
```

---

## Option 3: Einfacher Web-Hosting Service

### A) Netlify Drop (Sehr einfach!)
1. Gehe zu https://app.netlify.com/drop
2. Ziehe `privacy-policy-template.html` in den Browser
3. Netlify gibt dir eine URL: `https://random-name.netlify.app/privacy-policy-template.html`
4. ✅ Fertig!

### B) Vercel
1. Gehe zu https://vercel.com
2. "Add New" → "Project"
3. Lade die HTML-Datei hoch
4. Vercel gibt dir eine URL

---

## Für Google Play Console

### Nach dem Upload:

1. **Gehe zu Google Play Console**
2. **Deine App** → "Store presence" → "App content"
3. **Privacy policy** → "Start"
4. **Füge deine URL ein:**
   ```
   https://deinusername.github.io/plantry-privacy/privacy-policy-template.html
   ```
5. **Klicke "Save"**

---

## Wichtige Hinweise

### ✅ URL muss:
- Öffentlich zugänglich sein
- Keine Anmeldung erfordern
- HTTPS verwenden (nicht HTTP)
- Dauerhaft verfügbar sein

### ❌ Nicht akzeptiert:
- Dropbox/Google Drive Links (nicht dauerhaft öffentlich)
- PDF-Dateien (muss HTML/Webseite sein)
- Passwortgeschützte Seiten
- HTTP (unsicher)

## Schnellste Lösung: Netlify Drop

Wenn du es JETZT brauchst:

1. Öffne: https://app.netlify.com/drop
2. Ziehe `privacy-policy-template.html` ins Fenster
3. Warte 10 Sekunden
4. Kopiere die URL
5. Füge URL in Google Play Console ein
6. ✅ FERTIG!

**Zeit: ca. 2 Minuten**

---

## Test deine URL

Bevor du sie in Google Play einfügst:

1. Öffne die URL in einem privaten/Inkognito-Browser-Fenster
2. Prüfe, ob die Seite lädt
3. Prüfe, ob alles richtig angezeigt wird
4. Prüfe, ob die URL mit `https://` beginnt
